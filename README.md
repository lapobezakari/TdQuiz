# PolyQuiz — Plateforme Interactive de Quiz

Application web SPA développée avec React et Vite.js.  

## Installation et lancement

```bash
npm install
npm run dev
```

L'application sera accessible sur **http://localhost:5173**

## Structure du projet

```
brainwave/
├── public/
│   └── questions.json
└── src/
    ├── hooks/
    │   └── useFetch.js
    ├── context/
    │   └── UserContext.jsx
    ├── components/
    │   └── ProtectedRoute.jsx
    ├── pages/
    │   ├── Home.jsx
    │   ├── QuizEngine.jsx
    │   └── Results.jsx
    └── App.jsx
```
